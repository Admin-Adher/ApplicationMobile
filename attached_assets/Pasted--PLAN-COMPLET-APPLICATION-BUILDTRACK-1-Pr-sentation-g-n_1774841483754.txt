📘 PLAN COMPLET — APPLICATION BUILDTRACK

1️⃣ Présentation générale de l’application

Nom provisoire :

BuildTrack (modifiable)

Type d’application :

Application de gestion de chantier numérique destinée à remplacer :

Excel

papier

WhatsApp

emails dispersés


Elle centralise toutes les informations du chantier dans un seul outil.


---

🎯 Objectif principal

Créer une application permettant de :

suivre les réserves chantier

gérer les équipes

visualiser les plans

suivre l’avancement

centraliser les documents

améliorer la communication terrain


➡️ Le but est de remplacer un outil comme Dalux, mais adapté à tes besoins terrain.


---

👤 Utilisateurs cibles

Conducteurs de travaux

Ils pilotent le chantier.

Utilisation :

créer réserves

suivre avancement

consulter rapports



---

Chefs d’équipe

Ils exécutent les travaux.

Utilisation :

voir les tâches

ajouter photos



---

Bureau / direction

Ils supervisent.

Utilisation :

suivre indicateurs

consulter rapports

analyser performances
